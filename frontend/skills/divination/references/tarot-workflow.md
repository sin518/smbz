# Tarot Workflow (固定顺序)

## 0. Input Check

- **⏰ 先获取当前时间**（用于时间建议锚定）
- 所有输入均为可选：
  - `spreadType` — 牌阵类型：
    - `single` — 单牌（快速回答，默认）
    - `three-card` — 三牌（过去/现在/未来，标准场景）
    - `love` — 爱情牌阵（感情关系专题）
    - `celtic-cross` — 凯尔特十字（深度多维度解读）
  - `question` — 占卜问题（可选，建议聚焦一个主题）
  - `allowReversed` — 是否允许逆位（默认 true）
  - `seed` — 随机种子，固定可复现
- 牌阵选择策略：
  - 简单是非问题 → `single`
  - 带时间脉络的问题（过程/发展） → `three-card`
  - 感情相关 → `love`
  - 复杂人生大问题 → `celtic-cross`
- 抽牌前确认问题：聚焦一个主题（关系/事业/决策）
- 无明确问题时，默认"当前状态+下一步建议"

## 1. 抽牌层

- 调用 `tarot_draw`，得到：
  - `spreadName` → 牌阵名称
  - `cards[]` → 抽到的牌
  - `seed` → 本次种子（便于复现）
- 如需可复现，设置 `seed` 参数

## 2. 单牌语义层

- 逐张遍历 `cards[]`，提取：
  - `position` → 牌位含义（如"过去"、"现在"、"未来"、"阻碍"等）
  - `card.nameChinese` → 中文牌名
  - `card.keywords[]` → 关键词列表
  - `orientation` → 正位(upright) / 逆位(reversed)
  - `meaning` → 工具给出的牌义
- 逐张解释：
  - 牌位含义
  - 正逆位含义
  - 该牌对问题的直接指向

## 3. 牌阵关系层（Mandatory）

- 不只讲单牌，必须讲关系：
  - 支持/阻碍
  - 过去-现在-未来递进
  - 内因/外因区分

## 4. 行动建议层（Mandatory）

- 输出短期行动（3-7天）
- 输出中期行动（1-4周）
- 输出"不要做"的动作（避免误判）

## 5. 风险与边界层

- 塔罗用于决策辅助，不替代医学/法律/投资专业意见
- 高风险问题必须增加理性验证建议

## 6. Quick Template

1. 结论摘要（当前趋势）
2. 牌阵核心矛盾（支持 vs 阻碍）
3. 时间建议（短/中期）
4. 行动清单（做/不做）
